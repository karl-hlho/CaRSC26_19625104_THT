2a. **Git add** -> digunakan untuk perintah menambahkan file baru ke repository yang baru dipilih 
    
    Contoh penggunaan : 
    *menambahkan sebuah file *
    git add <namafile>
    
    *menambahkan beberapa file sekaligus (dalam hal ini 3 file)*
    git add <nama_file1> <nama_file2> <nama_file3>
    
    *menambahkan semua file atau semua perubahan yang terjadi*
    git add .
    
    **Git push** -> perintah yang digunakan untuk mentransfer perubahan file ke repository jarak jauh setelah melakukan perubahan
    Contoh penggunaan : git push <namaremote> <namabranch>
    
    **Git pull** ->  untuk mendapatkan update terbaru dari repository proyek pada gitlab, perintah ini dapat dijalankan seperti berikut
    Contoh penggunaan : git pull <namaremote> <namabranch>
    
    **Git merge** -> perintah dasar yang menggabungkan cabang aktif dengan cabang yang dipilih
    Contoh penggunaan : 
    #pindah ke branch master
    git checkout master
    #merge branch1 ke master
    git merge branch1
